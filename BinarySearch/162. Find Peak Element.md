## 找一个局部最大值

### 解法: 二分
如果满足条件则是; 否则查找左右中比当前数大的一边(两边都大则选一边)
```c++
class Solution {
public:
    int findPeakElement(vector<int>& nums) {
        int n = nums.size();
        if (!n) return 0;
        
        int low = 0, high = n-1;
        while (low < high) {
            int mid = low + (high - low) / 2;
            if ((mid == 0 || nums[mid] > nums[mid-1]) && (mid == n-1 || nums[mid] > nums[mid+1])) {
                return mid;
            }
            if (mid == 0 || mid < n-1 && nums[mid] < nums[mid+1]) low = mid + 1;
            else if (mid == n-1 || mid > 0 && nums[mid] < nums[mid-1]) high = mid - 1;
        }
        
        return low;
    }
};
```
