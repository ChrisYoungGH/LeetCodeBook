## 反转数
### 解法:
- 每次取低位追加到新数(避免用字符串转, 太慢)
- 注意溢出, 以及每一步计算时的类型转换
```c++
class Solution {
public:
    int reverse(int x) {
        if (!x) {
            return 0;
        }
        
        unsigned int ux = x > 0 ? x : (unsigned int)(-long(x));
        long res = 0;
        while (ux) {
            res = res * 10L + long(ux % 10);
            ux /= 10;
        }
        res = x > 0 ? res : -res;
        
        if (res > long(INT_MAX) || res < long(INT_MIN)) {
            return 0;
        }
        
        return int(res);
    }
};
```
