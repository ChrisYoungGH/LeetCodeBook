## 实现sqrt向下取整
### 解法
二分, 注意边界和表示范围(用/代替*防止int和long溢出)
- 注意:太大的数相乘long也可能会溢出

```c++
class Solution {
public:
    int mySqrt(int x) {
        if (x < 0) {
            return -1;
        }
        if (x <= 1) {
            return x;
        }
        
        int low = 1, high = x;
        while (low < high - 1) {
            int mid = low + (high - low) / 2;
            int val = x / mid;
            if (val == mid) {
                return mid;
            }
            if (val < mid) {
                high = mid - 1;
            }
            else {
                low = mid;
            }
        }
        
        return x / high >= high ? high : low;
    }
};
```
