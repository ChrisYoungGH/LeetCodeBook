最小栈: 使用一个附加的栈, 每次原栈入栈时, 附加栈入当前的最小值, 即将当前值与栈顶比较并入栈小者, 此时保证栈顶总是最小的. 取最小值时只需取栈顶. 若原栈弹出, 则附加栈相应弹出.
```c++
class MinStack {
public:
    /** initialize your data structure here. */
    MinStack() {
    }
    
    void push(int x) {
        data.push_back(x);
        if (mins.size() == 0) {
            mins.push_back(x);
        }
        else {
            int t = mins.back();
            mins.push_back(min(t, x));
        }
    }
    
    void pop() {
        if (data.size() > 0) {
            data.resize(int(data.size())-1);
            mins.resize(int(mins.size())-1);
        }
    }
    
    int top() {
        return data.back();
    }
    
    int getMin() {
        return mins.back();
    }

private:
    vector<int> data;
    vector<int> mins;
};
```
