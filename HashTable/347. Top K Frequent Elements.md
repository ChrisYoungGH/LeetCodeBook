## 找出现次数最多的k个数

### 解法:
先哈希记频数, 然后使用桶排序(因为最多出现n次, 故使用一个n+1大小的数组作为桶). 桶的下标为出现频数, 值为对应该频数的数的计数. 反向遍历桶以确定k个数的频数下界, 最后存储结果.

```c++
class Solution {
public:
    vector<int> topKFrequent(vector<int>& nums, int k) {
        int n = nums.size();
        
        unordered_map<int, int> hash;
        for (int num : nums) {
            hash[num]++;
        }
        
        vector<int> bucket(n+1, 0);
        for (auto it : hash) {
            bucket[it.second]++;
        }
        
        int count = 0;
        int low = n-1;
        while (count < k) {
            count += bucket[low];
            low--;
        }
        
        vector<int> res;
        for (auto it : hash) {
            if (it.second > low) {
                res.push_back(it.first);
            }
        }
        
        return res;
    }
};
```
