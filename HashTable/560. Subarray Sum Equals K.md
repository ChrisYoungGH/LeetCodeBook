## 数组连续子序列等于给定和的个数.

### 解法:
哈希存前缀和计数, 每次查看sum-k是否在哈希表中. 因sum(i,j)==k ==> sum(0,j)-sum(0,i-1)==k, i.e. preSum(j) - k = preSum(i-1)

```c++
class Solution {
public:
    int subarraySum(vector<int>& nums, int k) {
        int n = nums.size();
        if (!n) return 0;
        
        unordered_map<int, int> preSum;
        preSum[0] = 1;
        
        int res = 0;
        int sum = 0;
        for (int num : nums) {
            sum += num;
            if (preSum.find(sum - k) != preSum.end()) {
                res += preSum[sum - k];
            }
            preSum[sum]++;
        }
        
        return res;
    }
};
```
