## 长度为n的二叉搜索树数目
### 解法
动态规划:
- G(n): 长度为n的二叉搜索树数目
- F(i,n): 以i为根节点的二叉搜索树数目
- G(n) = sum(F(i,n)), i = 1, ..., n
- F(i,n) = 左*右 = G(i-1) * G(n-i)

```c++
class Solution {
public:
    int numTrees(int n) {
        vector<int> G(n+1);
        G[0] = G[1] = 1;
        for (int i = 2; i <= n; i++) {
            for (int j = 1; j <= i; j++) {
                G[i] += G[j-1] * G[i-j];
            }
        }
        
        return G[n];
    }
};
```
