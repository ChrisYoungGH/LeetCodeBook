## 罗马数字转整数
### 解法
遍历每个位置, 大于后一个则加, 否则减.
```c++
class Solution {
public:
    int romanToInt(string s) {
        map<char, int> rti = {
            {'I', 1},
            {'V', 5},
            {'X', 10},
            {'L', 50},
            {'C', 100},
            {'D', 500}, 
            {'M', 1000}
        };
        
        int res = 0;
        for (int i = 0; i < s.length() - 1; i++) {
            int curr = rti[s[i]];
            int next = rti[s[i+1]];
            if (curr < next) res -= curr;
            else res += curr;
        }
        
        return res + rti[s.back()];
    }
};
```
