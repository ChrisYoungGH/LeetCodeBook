## 能够到达的最远格
### 解法
遍历数组, 看之前最远是否到达当前格, 没到达则false; 根据当前格更新能够到达的最远格.

```c++
class Solution {
public:
    bool canJump(vector<int>& nums) {
        int n = nums.size();
        int farthest = 0;
        for (int i = 0; i < n - 1; i++) {
            if (farthest < i) {
                return false;
            }
            farthest = max(farthest, i + nums[i]);
            if (farthest >= n-1) {
                return true;
            }
        }
        return n <= 1;
    }
};
```
