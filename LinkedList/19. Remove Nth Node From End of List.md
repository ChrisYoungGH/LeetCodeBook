先求长度, 之后判断删除的位置:
- 链表外&rarr;不变
- 链表头&rarr;变换头
- 链表中间&rarr;变换指针, 需要记录curr和prev. 删除位置为len-n.

```c++
ListNode* removeNthFromEnd(ListNode* head, int n) {
    int len = 0;
    ListNode *curr = head;
    while (curr != NULL) {
        len++;
        curr = curr->next;
    }

    if (n > len || n <= 0) {
        return head;
    }
    if (n == len) {
        head = head->next;
    }
    else {
        curr = head;
        ListNode *prev;
        int currLen = 0;
        while (currLen < len - n) {
            prev = curr;
            curr = curr->next;
            currLen++;
        }
        prev->next = curr->next;
    }

    return head;
}
```
